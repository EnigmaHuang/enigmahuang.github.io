  • 【提瓦特测绘局】天空岛高度估算

    2021-05-07 |
    • { Genshin_Impact }
    • { geodetic }

    摘要：天空岛的方位引起了许多原神玩家好奇心。本文首次尝试利用钟离的岩脊作为标杆，构建长水平面基线和两组相似三角形，从而估算出天空岛的海拔高度约为 3000 米。同一方法测得寒天之钉顶部高度约为 553 米，表明此方法测得高度与实际高度相差约在 12%。

    关键词：相似三角形 岩脊 大地基线 高程传递
